Module: server_script2

This file contains a server script for a forum application. It handles incoming requests, saves new entries to a file, and generates an HTML response for the forum page. The server listens on port 8000.

Handles incoming requests.
Parameters:
Name Type Description
request IncomingMessage Input stream — contains data received from the browser, e.g,. encoded contents of HTML form fields.
response ServerResponse Output stream — put in it data that you want to send back to the browser. The answer sent by this stream must consist of two parts: the header and the body.
  • The header contains, among others, information about the type (MIME) of data contained in the body.
  • The body contains the correct data, e.g. a form definition.
